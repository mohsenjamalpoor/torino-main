Torino | Online Tour Reservation Application

A modern and responsive online tour reservation platform built with **Next.js**, allowing users to browse tours, authenticate with OTP, manage their profile, and complete tour reservations through a clean and user-friendly interface.

---

Features

- Browse available tours
- Tour details page
- OTP Authentication
- User profile management
- Checkout & Payment
- Responsive Design
- Server State Management with TanStack Query
- API Integration using Axios

---

 Technologies

- React.js
- Next.js (App Router)
- Tailwind CSS
- TanStack Query (React Query)
- Axios

---

  Installation

Clone the repository:

```bash
git clone https://github.com/mohsenjamalpoor/torino-main.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
